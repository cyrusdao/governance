## Migration

The governance platform has moved to the Pars DAO organization as part of the unified Persian DAO governance system.

**New Location**: [github.com/parsdao/pars.vote](https://github.com/parsdao/pars.vote)

**Live Site**: [pars.vote](https://pars.vote)

## Multi-Token Governance

The new platform supports all three Persian DAO tokens with equal voting power:

| Token | Project | Power |
|-------|---------|-------|
| CYRUS | Cyrus DAO | 1 Token = 1 Vote |
| MIGA | MIGA Protocol | 1 Token = 1 Vote |
| PARS | Pars Network | 1 Token = 1 Vote |
